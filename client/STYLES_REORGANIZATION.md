# Реорганизация стилей

## Текущая структура

```
src/
├── styles/
│   ├── base.css              # Базовые глобальные стили (125 строк)
│   ├── utilities.css         # Утилитарные классы (10 строк - очень маленький!)
│   └── bootstrap-overrides.css  # Переопределения Bootstrap (121 строка)
└── utils/
    └── animations.css        # Анимации (377 строк)
```

## Проблемы текущей структуры

1. **`utilities.css` слишком маленький** - всего 10 строк, логично объединить с `base.css`
2. **`animations.css` в `utils/`** - это стили, а не утилиты, логичнее в `styles/`
3. **Нет четкой категоризации** - все стили в одной папке без подкатегорий

## Варианты реорганизации

### Вариант 1: Минимальные изменения (рекомендуется)

```
src/
└── styles/
    ├── base.css              # Базовые стили + утилиты (объединены)
    ├── animations.css         # Анимации (перемещены из utils/)
    └── bootstrap-overrides.css  # Переопределения Bootstrap
```

**Преимущества:**

- Простота
- Минимум изменений
- Логичная структура

**Действия:**

1. Объединить `utilities.css` в `base.css`
2. Переместить `animations.css` из `utils/` в `styles/`
3. Обновить импорты в `App.css` и `App.js`

---

### Вариант 2: С подкатегориями

```
src/
└── styles/
    ├── base/
    │   ├── base.css          # Базовые стили
    │   └── utilities.css      # Утилитарные классы
    ├── animations/
    │   └── animations.css
    └── vendor/
        └── bootstrap-overrides.css
```

**Преимущества:**

- Четкая категоризация
- Легко масштабировать
- Профессиональная структура

**Недостатки:**

- Больше вложенности
- Нужно обновить больше импортов

---

### Вариант 3: Плоская структура с префиксами

```
src/
└── styles/
    ├── base.css
    ├── animations.css
    └── vendor-bootstrap.css  # Переименовать для ясности
```

**Преимущества:**

- Простота
- Понятные имена

---

## Рекомендация: Вариант 1

**Почему:**

- Минимум изменений
- Логичная структура
- `utilities.css` слишком маленький для отдельного файла
- `animations.css` логичнее в `styles/`, чем в `utils/`

## План действий (Вариант 1)

1. ✅ Объединить `utilities.css` в `base.css`
2. ✅ Переместить `animations.css` из `utils/` в `styles/`
3. ✅ Обновить импорты:
   - `App.css` - удалить импорт `utilities.css`, обновить путь к `animations.css`
   - `App.js` - обновить путь к `animations.css`
4. ✅ Удалить старые файлы
